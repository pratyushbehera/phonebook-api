Phonebook API documentation

This API allows storing updating and reading phone numbers from database.

This API has 3 endpoints

  • /getallphoneentries
    onFailure 
      {
    error: true,
    message: "Unable to fetch"
  }
    onSuccess 
    response - { 
  phoneEntries : {
      number: string,
      type: string,
      extn: string,
      city: string,
      state: string,
      desc: string,
      date: Date,
      isActive: Boolean  
    }
  }
  • /createphoneentry 
    request - { 
  phoneEntries : {
      number: string,
      type: string,
      extn: string,
      city: string,
      state: string,
      desc: string,
      date: Date,
      isActive: Boolean  
    }
  }
    onFailure 
        {
      error: true,
      message : "Phone entry addition failed"

    onSuccess

      message : "Phone entry addition successful"
